## Features

- **Greek Temple Aesthetic**: Marble pillars, divine lighting, and golden accents
- **Card Flip Effect**: Hover to reveal Korean translations on card backs
- **Particle System**: Floating golden particles creating an ethereal atmosphere
- **Lightning Effects**: Dramatic Zeus-inspired lightning flashes
- **Signature Dual Colors**: Each god has unique color combinations
- **Custom SVG Symbols**: Elaborate god-specific iconography
- **Responsive Design**: Adapts to desktop, tablet, and mobile
- **Scroll Animations**: Cards animate into view as you scroll

## Tech Stack

- HTML5 / CSS3 / JavaScript
- Google Fonts (Cinzel, Cormorant Garamond, Noto Sans KR)
- CSS 3D Transforms for card flip
- Canvas API for particle effects
- SVG for custom symbols and favicon
